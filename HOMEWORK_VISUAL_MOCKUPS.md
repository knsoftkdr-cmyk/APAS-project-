## Color Reference

### Timer Colors
- **Green (> 2:00)**: `text-green-600`
- **Yellow (1:00-2:00)**: `text-yellow-600`
- **Red (< 1:00)**: `text-red-600`

### Status Badges
- **Pending**: `border-border, text-muted-foreground`
- **Submitted**: `bg-emerald-500/15, text-emerald-700`

### Backgrounds
- **Timer Banner**: Color-coded with borders
- **Success**: `bg-emerald-50 dark:bg-emerald-900/20`
- **Warning**: `bg-yellow-50 dark:bg-yellow-900/20`
- **Alert**: `bg-red-50 dark:bg-red-900/20`

### Interactive Elements
- **Primary Button**: Blue background
- **Success Button**: Emerald background
- **Disabled**: Gray with no-cursor style

---

## Animations

- **Page Entry**: `animate-fade-in` on Start View
- **Timer**: Smooth counting without jitter
- **Button States**: Instant enable/disable
- **View Switch**: No transition (instant swap)

---

## Key Measurements (Tailwind)

- **Timer Font Size**: `text-4xl` (large and visible)
- **Question Number**: `text-sm` (readable)
- **Question Text**: `text-sm font-medium`
- **Answer Textarea**: `rows={6}` (One at a Time), `rows={3}` (All)
- **Spacing**: `space-y-4` (consistent vertical gaps)
- **Border Radius**: Default rounded corners
- **Shadow**: Subtle shadows on cards

---

## Interaction States

### Button States
```
NORMAL:      [Send ► Submit Homework]  (Clickable)
HOVERING:    [Send ► Submit Homework]  (Darker color)
LOADING:     [🔄 Submitting...]        (Disabled)
DISABLED:    [Submit Homework]          (Gray, no cursor)
```

### Input States
```
EMPTY:       (Placeholder showing)
FILLED:      (User text visible)
DISABLED:    (Gray background, read-only)
```

### Timer States
```
ACTIVE:      Counting down (00:XX:XX format)
PAUSED:      Never pauses (always counting)
COMPLETE:    Shows 00:00, frozen
```
